
# 🛡️ Sentinel Scan

**Sentinel Scan** is a futuristic, AI-powered cybersecurity tool designed to protect users from malicious links and AI-generated misinformation. Built with a focus on speed, safety, and community awareness.

## 🚀 Features

- **Unified Link Analysis:** Instantly scan suspicious messages for phishing, scams, and malware using Google's Gemini AI.
- **AI Image Detector:** A dual-method forensic tool that analyzes image metadata and visual artifacts to identify AI-generated content (DALL-E, Midjourney, etc.).
- **Community Threat Hub:** A moderated platform where users can report and view trending scams to keep others safe.
- **Interactive 3D UI:** A sleek, dark-themed interface featuring interactive Spline 3D models and high-performance animations.
- **Advanced Security Hub:** Access to a Web Extension, On-Call Detection, and Leaked DB Scanning.

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **AI Engine:** Genkit + Google Gemini 2.5 Flash
- **Database:** Firebase Firestore
- **Styling:** Tailwind CSS + ShadCN UI
- **Animations:** GSAP + Spline 3D
- **Icons:** Lucide React

## 📦 Deployment

### GitHub Upload
1. `git init`
2. `git add .`
3. `git commit -m "feat: initial commit"`
4. `git branch -M main`
5. `git remote add origin https://github.com/USERNAME/REPO.git`
6. `git push -u origin main`

### Changing Git Remote
If your project is already linked to another repository and you want to switch it:
1. **Check current remote:** `git remote -v`
2. **Set new URL:** `git remote set-url origin https://github.com/YOUR_USERNAME/NEW_REPO.git`
3. **Verify:** `git remote -v`
4. **Push:** `git push -u origin main`

### Vercel Hosting
1. Connect your GitHub repository to [Vercel](https://vercel.com).
2. Set the `GOOGLE_GENAI_API_KEY` environment variable in the Vercel dashboard.
3. Deploy!

## 🛡️ Security & Privacy
Sentinel Scan leverages real-time AI analysis. While highly accurate, always exercise caution with unknown links. Community reports are moderated to ensure high-quality, verified threat data.
